Lingering Questions

I am admittedly still unsure if I handled edge cases like "Scanlon" correctly. In this case, the second letter "c" is NOT followed by an "h". In this case, I assume you then move to the "additional characters" table and choose 3 for "c", but I'm not 100% sure. But, while my code gives "S365", this "G63" page (a PDF of which is on loc.gov) says it should be "S29".

Also, I think the "Cutter-Sanborn Three-Figure Author Tables (Swanson-Swift 1969 revision)" is the current "standard"? The codes produced by this tool do not match those used by Pratt libraries, so something is off.

I can't seem to find the Swanson-Swift 1969 revision online, but Pratt seems to have a copy at the Brooklyn campus library.
