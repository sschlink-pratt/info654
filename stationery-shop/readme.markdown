### Other fun things I did:
* Use [CSS's `max-width` property](https://www.w3schools.com/css/css_max-width.asp) to make various HTML elements, like tables, adapt to narrower screens better.
* Product images expand and tilt when hovered over, which I adapted from [this tutorial](https://www.30secondsofcode.org/css/s/image-hover-rotate/)

## Other sources consulted

I consulted this Stack Overflow Q&A for help with making a table with <div> tags on `index.html`: [https://stackoverflow.com/a/48570166](https://stackoverflow.com/a/48570166). Basically, I set the `div`s that represent the rows of the table to `display: flex;`, a ["layout model for arranging items (horizontally or vertically) within a container, in a flexible and responsive way"](https://www.w3schools.com/css/css3_flexbox.asp).

To create the "zebra-striped" table on the `products/mechanical-pencil.html` page, I used [this W3 demo page](https://www.w3schools.com/css/tryit.asp?filename=trycss_table_striped) as a guide.
